## MSX-YAMAHA-UCNM-01
adaptator only for Yamaha SFG-01 & SFG-05

Adapator from 60 pin SFG-01 or SFG-05 to 50 pin MSX

* you need a PinHeader 1x03 P2.54mm Horizontal

PCB with Electroless Nickel / Immersion Gold is better
